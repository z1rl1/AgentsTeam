## 1. Покажи здоровье системы
```bash

```

## 2. Покажи созданные игры
```bash
python3 /root/.openclaw/workspace/skills/game-catalog/scripts/game_catalog.py
```

- Модель отвечает?
- Surge доступен?

После проверки — жди сообщений от пользователей.
