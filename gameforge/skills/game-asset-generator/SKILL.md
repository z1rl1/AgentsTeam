# Game Asset Generator Skill

## Задача
Создаёт визуальные ассеты для игр без внешних изображений.

## Инструменты
- **CSS shapes** — геометрические персонажи и объекты
- **SVG inline** — векторные спрайты прямо в HTML
- **Unicode/Emoji** — быстрые иконки и символы
- **Canvas drawing** — процедурная графика через JS
- **CSS animations** — анимированные спрайты

## Примеры
- Пиксельный персонаж через CSS grid
- SVG зомби/робот/космический корабль
- Particle effects через Canvas
- Animated background через CSS keyframes

## Алгоритм
1. Определи что нужно нарисовать
2. Выбери подходящий метод (CSS/SVG/Canvas)
3. Напиши код ассета
4. Встрой в существующую игру
5. Задеплой
